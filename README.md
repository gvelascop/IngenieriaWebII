# 🎉 EventHub

Interfaz web para una plataforma digital de gestión y promoción de eventos (conciertos, conferencias, talleres y festivales), desarrollada como actividad de la asignatura **Ingeniería Web II**.

## Tecnologías utilizadas

- **HTML5** — estructura semántica del sitio
- **CSS3** — estilos personalizados (`css/style.css`)
- **Bootstrap 5.3.7** — framework de estilos para diseño responsive y componentes UI
- **JavaScript (Vanilla)** — interacciones (`js/script.js`)

## Estructura del proyecto

```
IngenieriaWebII-master/
├── index.html          # Página principal
├── css/
│   └── style.css        # Estilos personalizados
├── js/
│   └── script.js         # Lógica de interacción
├── img/                  # Imágenes de los eventos
└── README.md
```

## Funcionalidades

- Navbar responsive con menú colapsable en móviles y resaltado automático de la sección visible.
- Banner de bienvenida (hero) con llamado a la acción.
- Sección de estadísticas (eventos publicados, asistentes, categorías, organizadores aliados).
- Sección de eventos destacados con 6 tarjetas (fecha, lugar, precio y categoría de cada evento).
- Filtro de eventos por categoría (Música, Tecnología, Arte, Educación, Deportes).
- Sección de testimonios de asistentes y organizadores.
- Formulario de contacto con validación en tiempo real (nombre, correo y mensaje).
- Diseño 100% responsive: adaptado a computador, tablet y celular.

## ▶️ Cómo ejecutar el proyecto

1. Cloná o descargá este repositorio.
2. Abrí el archivo `index.html` en tu navegador (no requiere instalación ni servidor).
